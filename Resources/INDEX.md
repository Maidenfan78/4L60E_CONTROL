# Resources Index

This folder is reference material (PDFs, wiring diagrams, connector pinouts, photos, board notes).

## How to use

- If you’re using QMD (or any search tool), it will index **markdown/txt** best.
- For PDFs/images, consider adding **text sidecars** next to the file (same name + `.txt`) for searchability.

## Subfolders

- `4l60e/` — transmission reference docs (manuals/specs/wiring diagrams/pinouts)
- `pi_zero_2w/` — Raspberry Pi Zero 2W reference (pins etc)

See also: `resources/4l60e/INDEX.md`
