# Firmware Workspace

This folder contains the Teensy 4.1 firmware sources for the 4L60-E manual-mode controller.

## Structure

- `src/` - Application source files
- `include/` - Public headers shared across modules
- `config/` - Build- or runtime-config headers
- `lib/` - Local libraries (if needed)
- `test/` - Unit or host-side tests
- `docs/` - Firmware-specific notes
- `scripts/` - Helper scripts for build/flash
